# Trade Analytics Copilot — Design Document

A professional trading analytics platform modeled after TraderSync. This document captures the full UI design across all 11 shipped features, with annotated screenshots from the mockup canvas.

---

## Feature 1 — Dashboard

P&L summary, win rate, profit factor, expectancy, equity sparkline, and open position summary. The command center a trader sees on login.

![Dashboard](screenshots/01-dashboard.jpg)

---

## Feature 2 — Trade Ledger with R-Multiple & Live P&L

Full trade history with color-coded R-Multiple column (+2.1R / +0.8R / -0.7R). Open positions show a pulsing green dot with the current live price and P&L from Alpaca's quote API refreshed on each page load.

![Trade Ledger](screenshots/02-trades-r-multiple.jpg)

---

## Feature 3 — Per-Trade Candlestick Chart

OHLCV candlestick chart rendered with TradingView `lightweight-charts`. Entry marked with an amber Buy arrow and a labeled dashed price line; exit marked with a green Sell arrow and price line. Stop loss and take-profit levels drawn as dashed overlays. R-Multiple card is highlighted with an orange ring.

![Candlestick Chart](screenshots/03-candlestick-chart.jpg)

---

## Feature 4 — MAE / MFE Trade Efficiency Bar & Screenshot Gallery

The excursion bar spans from the worst intra-trade drawdown (MAE, red zone) to the best possible exit (MFE, green zone), with a dot at the actual exit to show capture efficiency. Screenshots are stored as base64 JSON in the database and displayed in a 3-column gallery with drag-and-drop upload.

![MAE/MFE and Screenshot Gallery](screenshots/04-maemfe-screenshots.jpg)

---

## Feature 5 — Analytics: Equity Curve, Time Analysis & Setup Performance

Cumulative equity curve with 1W/1M/3M/6M/1Y/All range selectors, max drawdown annotation, plus breakdowns by time-of-day and setup type.

![Analytics Equity Curve](screenshots/05-analytics-equity.jpg)

---

## Feature 6 — Journal with AI Semantic Search

Journal log with date/tag filters and a natural-language search bar backed by OpenAI `text-embedding-3-small` embeddings stored in Pinecone. Entries show trade symbol, side, P&L, tags, mood emoji, and a pre/post trade note preview. Falls back to PostgreSQL `LIKE` if Pinecone is unavailable.

![Journal](screenshots/06-journal-search.jpg)

---

## Feature 7 — AI Coaching Alerts

Real-time coaching feed generated by the backend from trade history. Alerts are categorised as Critical / Warning / Info and include specific evidence (e.g. "You held TSLA 4.2R beyond your defined stop"). Each alert links to the triggering trade for immediate review.

![Coaching Alerts](screenshots/07-coaching-alerts.jpg)

---

## Feature 8 — Weekly AI Review

AI-generated weekly digest summarising net P&L, win rate delta vs prior week, and pattern-detected insights ("Your 10AM–11AM window is 3× more profitable than any other hour"). Side-by-side metric table shows this week vs last week across P&L, win rate, avg R, and trade count.

![Weekly Review](screenshots/08-weekly-review.jpg)

---

## Feature 9 — Alpaca Broker Sync (203 real trades, FIFO matching)

Alpaca paper-trading sync via a FIFO buy→sell matching engine. 203 round-trip trades imported. Sync history table shows job ID, start time, duration, trades fetched, and status. Failed jobs expose the error inline with a Retry action.

![Broker Sync](screenshots/09-broker-sync.jpg)

---

## Feature 10 — Portfolio: Live Positions from Alpaca

Real-time portfolio view pulling live positions from Alpaca's REST API. Shows total equity, today's P&L, open position count, and buying power across multiple accounts. Each open position row shows current market value and a Close action.

![Portfolio](screenshots/10-portfolio.jpg)

---

## Feature 11 — Strategy Drift Analysis

Drift score (0–100) measures how far current trading behaviour has deviated from the trader's most profitable historical period. Breakdown across three dimensions: Setup Adherence, Risk Discipline, and Time-of-Day Compliance. A "Recalibrate to Best Period" action sets a new baseline.

![Strategy Drift](screenshots/11-strategy-drift.jpg)

## Canvas Board

The full design progression lives on the canvas across **7 phases**:

| Phase | Screens |
|---|---|
| 1 — Foundation | Login · Dashboard |
| 2 — API Ingestion | Broker Connections · Alpaca Sync · Trades from Alpaca |
| 3 — Analytics Engine | Equity Curve · Time Analysis · Setup Performance |
| 4 — Tagging & Journaling | Trade Detail · Tag Management · Journal Log |
| 5 — Insight Engine | Weekly Review · Coaching Alerts · Strategy Drift |
| 6 — Broker Expansion | Multi-broker · Sync Orchestration · Portfolio |
| 7 — TraderSync Parity | Candlestick Chart · MAE/MFE Efficiency · R-Multiple + Live P&L |

## Tech Stack Summary

| Layer | Technology |
|---|---|
| Frontend | React 18 + Vite + Tailwind + shadcn/ui |
| Backend | Python 3.11 / FastAPI / Uvicorn |
| Database | PostgreSQL + SQLAlchemy ORM |
| Charts | TradingView `lightweight-charts` |
| AI / Search | OpenAI `text-embedding-3-small` + Pinecone serverless |
| Broker | Alpaca paper trading API (FIFO matching engine) |
| Auth | Session cookies (starlette `SessionMiddleware`) |
